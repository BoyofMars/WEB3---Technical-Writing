## 3. How NFTs Are Created (Minting)
NFT minting is the process of writing token data to a blockchain through a smart contract.

It includes:
- Token creation  
- Metadata assignment  
- Wallet linking  
- Payment of gas fees  

---

## 4. Smart Contracts Explained
NFTs use token standards such as:
- ERC-721  
- ERC-1155  

Smart contracts define:
- Ownership  
- Transfer rules  
- Royalties  
- Metadata locations  

---

## 5. NFT Metadata
Metadata contains:
- Name  
- Description  
- Image URL  
- Traits  

It can be stored on:
- On-chain  
- IPFS  
- Traditional servers
